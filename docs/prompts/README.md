## Рекомендуемый рабочий процесс

Перед запуском любого агента:
1. Прикрепить `00_PROJECT_CONTEXT.md` (обязательно)
2. Прикрепить `35_AGENT_HANDOFF_CURRENT.md` (для кодовых задач)
3. Не давать агентам старые промпты 10/11/20/21/22/30–34 без предупреждения об архиве

```
        ┌───────────────────────────────────────────────┐
        │  ZCode (архитектор + координатор)              │
        │  • ставит окружение, SDK, сборку               │
        │  • пишет математику/домен                       │
        │  • ревьюит код OpenCode и интегрирует          │
        │  • собирает финальный APK/AAB                   │
        └───────────────┬───────────────────┬───────────┘
                        │                   │
            код-ревью ← │                   │ → вставка контента
                        ▼                   ▼
        ┌───────────────────────┐   ┌───────────────────────┐
        │  OpenCode (локально)   │   │  Gemini (онлайн)       │
        │  • генерит Flutter-код │   │  • статьи энциклопедии │
        │  • виджеты, провайдеры │   │  • EULA / privacy      │
        │  • flutter analyze ✓   │   │  • листинги сторов     │
        └───────────────────────┘   └───────────────────────┘
```
