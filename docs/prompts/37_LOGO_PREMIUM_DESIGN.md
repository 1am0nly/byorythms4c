# 37 — Промпт: логотип + премиальный дизайн

## Задача A — Логотип
Приложение «Биоритмы», 4 цикла: физический (мятный #7FD1AE), эмоциональный
(лавандовый #B5A8D5), интеллектуальный (персиковый #F2C2A0), интуитивный
(бирюзовый #8CD8F5). Минимализм, calm/zen.

Нужно 3 концепта иконки (1024×1024, без текста, узнаваема в 48×48):
- скруглённый квадрат ИЛИ круг
- 4 волны/дуги, переплетающиеся, в 4 цветах цикла
- работает на светлом и тёмном фоне (Android adaptive icon: foreground+background слои отдельно)

Формат ответа на каждый концепт:
```
КОНЦЕПТ N: [название]
Форма/фон/элемент/цвета/настроение
```

После выбора — сгенерировать PNG 1024×1024, прогнать через
`flutter_launcher_icons` (добавить в pubspec.yaml, запустить
`dart run flutter_launcher_icons`).

## Задача B — Премиальный дизайн (нужна конкретика от Антона перед стартом)
Уточнить у Антона до начала:
1. Какие экраны — home, paywall, settings?
2. Референс-приложение со стилем, который нравится?
3. Что не устраивает — плоскость/тени, типографика, стандартные Material-виджеты?

Направления по умолчанию, если ответа нет:
- Тени/глубина: `elevation` на карточках → мягкие `BoxShadow` вместо плоских `Card`
- Градиенты на paywall-кнопках вместо плоской заливки `FilledButton`
- Кастомный `AppTextTheme` — не дефолтный Roboto, подключить Google Font
  (например Manrope/Inter) через `google_fonts` пакет
- Skeleton-загрузка вместо `CircularProgressIndicator` на paywall
- Проверить отступы по 8px-сетке везде единообразно

Начинать с paywall_screen.dart — это витрина монетизации.
